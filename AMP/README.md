# AMP for Typecho
 A typecho plugin for Google AMP

开启后对Google的搜索结果更加友好。

之前[直接通过暴力修改模板][1]实现的，结果发现有不少TX需要这个功能，所以整理了一下做成插件，方便使用。

目前仅支持MarkDown书写的日志。

## 功能

自动生成符合Google AMP标准的AMP页面，并与标准页面建立关联。


## 安装

将文件夹重命名为`AMP`，然后拷贝至`usr/plugins/`下，最后在后台->插件处安装。

## 升级方法

**请先禁用插件后再升级**

## 使用

到插件后台设置一下默认LOGO和默认图片即可。
